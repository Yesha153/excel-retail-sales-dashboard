# 📊 Retail Sales Dashboard (Excel Project)

## 📌 Overview
This project analyzes retail sales data using Excel to uncover insights on revenue trends, customer behavior, and product performance.

---

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- VLOOKUP
- Data Cleaning
- Slicers (Interactive Dashboard)

---

## 📊 Key Features
- Interactive dashboard with filters
- Revenue analysis by category and store
- Customer payment behavior analysis
- Top 10 products by revenue
- Sales calculation using formulas

---

## 📈 Key Insights
- Fashion and Electronics generate the highest revenue
- Credit Card and Cash are the most used payment methods
- Store performance is consistent across locations
- A few products contribute significantly to total sales
